Privacy Policy

Last updated: July 9, 2026

Short version: everything stays on your Mac. Flit is a local desktop web browser. It has no accounts, no servers, and no interest in your data.

Data storage

All browsing data lives locally on your computer:

  • Sessions and logins — each account's cookies and site storage are kept in an isolated local session partition.
  • History, bookmarks, downloads, and preferences — stored in local files on your Mac.

Flit never uploads, syncs, or transmits this data anywhere. Removing an account wipes its session data from disk.

No telemetry

Flit contains no telemetry, analytics, crash reporting, or tracking of any kind.

Network connections Flit makes

The only network traffic Flit generates is:

  • The sites you visit — requested directly, like any browser. Each site's own privacy policy applies to what you do there.
  • Search suggestions — as you type in the address bar, the query is sent to your chosen search engine (Google, DuckDuckGo, or Bing) to fetch live suggestions.
  • Favicons and account avatars — small image fetches used to draw site and account icons.
  • Update checks — Flit checks GitHub Releases for new versions and downloads updates from there.

Extensions

If you install Chrome extensions, they are downloaded from the Chrome Web Store and update from there. Extensions are third-party software with their own privacy policies.

Questions

Flit is open source (GPL-3.0), so you can verify all of the above in the source code. Questions or concerns: GitHub Issues.
